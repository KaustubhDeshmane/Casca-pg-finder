## 📖 Overview

Casca is a web-based accommodation platform developed to bridge the gap between students, working professionals, and property owners. The platform provides a user-friendly interface for discovering rental properties, viewing detailed listings, and interacting with landlords.

This project was built using HTML, CSS, JavaScript, PHP, and MySQL, integrating an interactive map for enhanced property discovery.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/KaustubhDeshmane/Casca-pg-finder.git
```

### Navigate to the Project

```bash
cd Casca-pg-finder
```

### Run Locally

* Place the project inside your XAMPP/WAMP `htdocs` directory.
* Start Apache and MySQL.
* Import the required MySQL database (if applicable).
* Open your browser and navigate to:

```text
http://localhost/Casca-pg-finder/
```
