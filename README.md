# 🏠 Casca – PG & Hostel Finder

A responsive web application designed to simplify the process of finding and managing PGs and hostels. Casca enables users to browse properties, explore locations through interactive maps, submit reviews and support tickets, while allowing landlords to manage property listings efficiently.

---

## 📖 Overview

Casca is a web-based accommodation platform developed to bridge the gap between students, working professionals, and property owners. The platform provides a user-friendly interface for discovering rental properties, viewing detailed listings, and interacting with landlords.

This project was built as a full-stack web application using HTML, CSS, JavaScript, PHP, and MySQL.

---

## ✨ Features

### 👤 User Features

* Browse available PGs and hostels
* Interactive property listings
* Property search interface
* View property details
* Submit reviews
* Raise support tickets
* User authentication

### 🏡 Landlord Features

* Add new property listings
* Upload multiple property images
* Manage listed properties
* View customer reviews
* View support tickets

### 🌍 Additional Features

* Responsive user interface
* Interactive maps using Leaflet.js
* Clean and intuitive navigation
* Image upload support
* Contact page for team communication

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/KaustubhDeshmane/Casca-pg-finder.git
```

### Navigate to the project

```bash
cd Casca-pg-finder
```

### Run locally

* Place the project inside your XAMPP/WAMP `htdocs` directory (if using PHP).
* Start Apache and MySQL.
* Open your browser and navigate to:

```text
http://localhost/Casca-pg-finder/
```
